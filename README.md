# Init project

> Install packages

```
npm i
```

> Copy .env.example to .env & fill credentials

```
cp .env.example .env
```

- `PORT`: port for local development
- `API_URL`: api url for requesting queries to backend

> Run local severr for development

```
npm run dev
```

> Build your project and minify for production

```
npm run build
```
